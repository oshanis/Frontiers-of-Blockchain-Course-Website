# Assignments

## Assignment Submission

All assignments must be submitted through **Submitty**: [https://submitty.cs.rpl.edu/courses/s26/csci4190](https://submitty.cs.rpl.edu/courses/s26/csci4190)

## General Guidelines

### Submission Requirements

- **Deadline:** Assignments are due by the specified date and time on Submitty
- **Format:** Follow the submission format specified for each assignment
- **Documentation:** Include clear comments and documentation in code

### Collaboration Policy

- You may discuss general concepts and approaches with classmates
- All code and solutions must be your own work
- Cite any external resources or references used, including GenAI tool use
- Violations will be handled according to academic integrity policies
